# OPIM 5512 — Applied Data Science — Course Notebooks

Standardized, ready-to-run notebooks for OPIM 5512 — Applied Data Science.

- Each notebook has a banner, a title, and an **Open in Colab** badge — click it and **Runtime → Run all**.
- Data loads from stable links (no Drive mounting, no uploads).
- Install deps with `pip install -r requirements.txt`.

Organized by module. Original source notebooks remain in their respective `*Files` repos.
